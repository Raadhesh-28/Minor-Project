# Indian Traditional Painting Classification

## Project Overview
This project focuses on classifying various Indian traditional painting styles using machine learning techniques. We explore the use of Convolutional Neural Networks alongside hybrid feature extraction to identify unique structural and stylistic traits of regional art forms.

## Dataset Description
The dataset contains images of multiple Indian traditional painting styles, properly categorized and mapped.
